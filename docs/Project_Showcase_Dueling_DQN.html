Deep Reinforcement Learning

多无人机突发事件临机决策

基于 Dueling DQN + 优先经验回放 (PER) 的 V2 轮询 MDP 架构，实现 4 机协同搜索的实时动态重规划

◆ 算法核心架构

1. Dueling DQN（竞争深度 Q 网络）

传统 DQN 直接输出每个动作的 Q 值。而 Dueling DQN 将 Q 值评估拆分为两条独立路径：

  • 状态价值头 ：评估当前局面的好坏程度。
  • 动作优势头 ：评估选择动作 相对其他动作的额外收益。

这种分离设计使网络在不关心具体动作时（如状态极为优越或极差），能更精准地学习状态本身的价值，极大提高训练稳定性与收敛速度。

2. 优先经验回放 (PER)

在复杂约束任务（如绕飞禁飞区）中，包含"教训"或"惊喜"的样本更加珍贵。本项目通过 二叉线段树 实现高效 PER：

  • 根据 TD-Error 决定采样优先级——误差越大，越需频繁回放。
  • 引入重要性采样权重消除分布偏差，参数 从初始值线性退火至 ，确保训练末期无偏。

◆ MDP 建模（116 维状态 · 20 维动作）

1. 状态空间 — 116 维特征矩阵

特征群维度描述
无人机状态矩阵124 架 UAV 的 [x, y, range_remaining] 归一化坐标
全局目标态势100 最大 20 个目标槽位，每槽 5 维：[cx, cy, weight, mask, competitive_ratio]
competitive_ratio = 友机最近距 / 本机距，显式引导网络防抢夺。
决策者身份4当前决策 UAV 的 One-Hot 编码

2. 动作空间 — 20 维离散动作

采用以目标为导向的决策：N_MAX = 20。动作 代表当前 UAV 选择飞往全局目标池中的对应索引。结合启发式动作掩码，物理剔除超限或半途锁定的动作。

4. 奖励设计 — 深度解析

A. 单步即时奖励
  • 目标权重收益：驱动网络优先飞往高危目标。
  • 距离折损：惩罚舍近求远的行为。
  • 抢夺惩罚：若目标明显位于友机附近（越区抢夺），施加 惩罚，迫使机群学会"礼让"。
B. 回合终止奖励
  • 完美收官：所有任务分配完毕，全局通关奖励 。
  • 遗漏惩罚：任务烂尾（坠毁或超限未分配），扣减 。

◆ 轮询决策循环逻辑

放弃死板的同步决策，采用自然贴合真实世界时序的 V2 轮询决策机制：

  1. 环境重置与校验：reset_v2 随机选择存活 UAV，通过 while checked < 4 循环审查其是否拥有合法动作。若无活可干，自动流转至下一架。
  2. 半途锁定：若突发事件发生时某 UAV 正在螺旋搜索中途（progress_km > 0），强制生成唯一 Action Mask 锁定当前目标，防止沉没成本浪费。
  3. 序列化状态推进：当前 UAV 执行动作后目标被消耗、状态刷新，随后指针 (idx + 1) % 4 流转。
  4. 阻断误差泄露：将下一状态的合法动作掩码存入 PER 缓存，在 Bellman 方程中物理阻断非法动作干扰，确保平稳收敛。
